SupportSys is a modern support ticketing system where users can register, submit issues, and receive responses from admins. It simulates a real IT help desk, fully developed with PHP, MySQL, Bootstrap 5 and responsive design.

🚀 Features:
- User/Admin login system
- Ticket submission by category
- Admin dashboard with replies + status updates
- Responsive landing page
- Dark mode toggle
- Animated counters
- Image carousels for UI + testimonials
- FAQ section
- Floating chat-style help modal

💻 Technologies Used:
- PHP 8.1
- MySQL (via HeidiSQL)
- HTML5 + CSS3 + Bootstrap 5
- JavaScript + Typed.js
- Visuals from UnDraw & RandomUser

🎯 Learning Goals Achieved:
- Dynamic front-end with animation
- Secure user authentication
- Database CRUD (tickets, users, responses)
- Visual polish and modern UX principles
